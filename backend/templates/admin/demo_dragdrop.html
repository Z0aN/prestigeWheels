Где найти функциональность

В Django Admin панели:
  • Cars → Редактирование автомобиля → Inline секция "Car images"
  • Car images → Добавить/редактировать изображение
  • Массовая загрузка → Actions в списке автомобилей
В API endpoints:
  • POST /api/cars/{id}/upload-image/ - одиночная загрузка
  • POST /api/cars/{id}/upload-images/ - массовая загрузка
  • GET /api/cars/{id}/images/ - получить изображения

Как использовать

Войдите в Django Admin

Перейдите по адресу /admin/ и войдите с правами администратора

Откройте раздел Cars

Выберите автомобиль для редактирования или создайте новый

Найдите секцию Car images

Прокрутите вниз до inline секции с фотографиями автомобиля

Используйте Drag & Drop

Перетащите изображение в специальную зону или нажмите для выбора файла

Заполните метаданные

Добавьте название, описание, настройте порядок отображения

Сохраните изменения

Нажмите "Save" для сохранения всех изменений

Технические детали

Файловые ограничения:
  • Максимальный размер: 5MB
  • Форматы: JPEG, PNG, GIF
  • Автоматическое изменение размера
  • Валидация на стороне клиента и сервера
Технологии:
  • JavaScript - обработка событий
  • CSS3 - стили и анимации
  • Django Forms - валидация
  • HTML5 File API - работа с файлами
Превью интерфейса админки:

Перетащите изображение сюда или нажмите для выбора
Поддерживаются: JPEG, PNG, GIF (до 5MB)
